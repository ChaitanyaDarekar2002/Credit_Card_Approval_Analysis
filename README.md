# Credit Card Approval Analysis

A data analysis project exploring credit card approval patterns across 500 applicants using Python, Excel, and data visualization.

---

## Dataset Overview

| Field | Details |
|---|---|
| Records | 500 applicants |
| Approved | 273 (54.6%) |
| Rejected | 227 (45.4%) |
| Features | 17 columns |

**Key columns:** `Application_ID`, `Age`, `Gender`, `City`, `Annual_Income`, `Employment_Type`, `Credit_Score`, `Debt_to_Income_Ratio`, `Approval_Status`, `Credit_Limit_Granted`

---

## Key Findings

- **Credit score** is the strongest predictor — applicants above 750 have a 95%+ approval rate
- **Income below ₹3L** results in zero approvals
- **Debt-to-income ratio** of rejected applicants is 2.4× higher than approved (1.31 vs 0.55)
- **Government employees** have the best approval rate (61%); **Freelancers** the lowest (48%)
- **Hyderabad** leads city-wise at 70% approval; **Mumbai** is lowest at 39%
